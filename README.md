# ClubeDaLeitura

![](https://imgur.com/9eKuNz8.gif)

## Introdução 
 
 Trata-se de um programa criado com o intuido de simular um clube de leitura, o qual possui amigos que imprestam livros uns aos outros.
 
 ## Funcionalidades
 
 - Todos os módulos possuem o CRUD de acordo com seu propósito e atuação.
 - O Programa foi criado seguindo os métodos:
 > Orientação a Objetos: Facilitando a compreenção e edição do mesmo.
 
 > Encapsulamento: Protege os dados que estão sendo utilizados no programa.

 > Herança e Polimorfismo: Reutilização do código, facilitando a compreenção e futuras edições.
